CONTOH PENGGUNAAN
================================================================================

RENAME CERTIFICATES:
  1. Open ‚Üí Certificate Tool.exe
  2. Tab "Rename Files"
  3. Folder Sumber: rsrc (ada sertif-1186.pdf, sertif-1187.pdf, dst)
  4. File Excel: acara.xlsx (ada kolom: nama, divisi)
  5. Format Output: {nama}_Peserta {divisi}
  6. Klik "Mulai Rename"
  
  HASIL:
  Budi Santoso_Peserta Acara.pdf
  Siti Nurhaliza_Peserta Danus.pdf
  Ahmad Wijaya_Peserta Humas.pdf


SPLIT PDF (2000 halaman ‚Üí 2000 file):
  1. Tab "Split PDF"
  2. Pilih file: sertifikat peserta seminar.pdf
  3. Folder Output: split_output
  4. Format: peserta-{nomor}
  5. Klik "Mulai Split"
  
  HASIL:
  split_output/peserta-1.pdf
  split_output/peserta-2.pdf
  ... (sampai peserta-2000.pdf)


MERGE PDF:
  1. Tab "Merge PDF"
  2. Pilih folder: split_output
  3. Output name: merged.pdf
  4. Klik "Mulai Merge"
  
  HASIL:
  C:\sertifikat eea\merged.pdf

TROUBLESHOOTING
================================================================================

ERROR: "ModuleNotFoundError"
SOLUSI: pip install pandas openpyxl PyPDF2

ERROR: "File tidak ditemukan"
SOLUSI: Pastikan file PDF ada di folder sumber dengan nama yang sesuai

ERROR: "Kolom tidak ditemukan"
SOLUSI: Periksa nama kolom di Excel (must match placeholder exactly)
  ‚úì {nama} - pastikan ada kolom "nama" di Excel
  ‚úì {divisi} - pastikan ada kolom "divisi" di Excel

ERROR: ".exe tidak bisa dibuka"
SOLUSI: 
  1. Coba jalankan as Administrator
  2. Disable antivirus sementara
  3. Atau jalankan dari Python source code
